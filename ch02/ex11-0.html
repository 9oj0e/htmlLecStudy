form태그

속성 설명
action 데이터의 전송 대상 페이지를 지정합니다.
method 데이터의 전송 방법을 지정합니다. 주로 get또는 post를 사용합니다.
name 입력 양식의 이름을 지정합니다.

한 줄 텍스트 입력란의 속성값

속성값 설명
text 기본적인 텍스트(초깃값)
search 검색 전용 텍스트
email 메일 주소
tel 전화번호
url 웹 사이트의 url

라디오 버튼을 만드는 radio

속성 설명
name 라디오 버튼의 이름
value 전송할 선택지의 값
checked 처음부터 선택돼 있게 하고 싶을 때 지정

속성 설명
name 체크 박스의 이름
value 전송될 선택지의 값
checked 처음부터 선택돼 있게 하고 싶을 때 지정
